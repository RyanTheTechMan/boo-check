# Boo Check Privacy Policy

Last updated: June 10, 2026

Boo Check is an unofficial Chrome extension for importing selected media from web pages into a user-configured, self-hosted Blombooru instance.

## Data handled by the extension

Boo Check may handle the following data when you use the extension:

- Your Blombooru base URL and API key.
- The media file you choose to import.
- Page, source, preview, and media URLs related to the selected import.
- Metadata you review or edit before import, such as artist, rating, tags, and media dimensions.
- Click or right-click events used to select media when multi-add capture is enabled.

## How data is used

This data is used only to prepare and send imports to the Blombooru instance configured by the user. The extension stores settings locally in Chrome storage and stores temporary import drafts and queues in Chrome session storage.

Boo Check does not sell user data. Boo Check does not transfer user data to third parties, except when sending the selected import data to the Blombooru server configured by the user or when fetching selected media from its original website.

## Remote code

Boo Check does not execute remote JavaScript or WebAssembly. Extension code is included in the extension package.

## Contact

For questions or issues, contact the extension developer through the support channel listed on the Chrome Web Store item page.
